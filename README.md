## Project Overview

**Aadrila AI** is a Next.js application that leverages artificial intelligence to provide intelligent solutions. This project demonstrates modern web development practices using the Next.js framework with TypeScript.

## Features

- Built with Next.js 14+ and React
- TypeScript for type safety
- Responsive design
- AI-powered functionality
- Optimized font loading with next/font

## Prerequisites

- Node.js 18.17 or later
- npm, yarn, pnpm, or bun package manager

## Project Structure

```
aadrila-ai/
├── app/              # Next.js app directory
├── public/           # Static assets
├── package.json      # Project dependencies
└── README.md         # This file
```

## Development

Modify `app/page.tsx` to customize the main page. The development server provides hot-reload functionality.

## Technologies Used

- **Framework:** Next.js
- **Language:** TypeScript
- **Font:** raleway (via next/font)
- **Styling:** CSS Modules / Tailwind CSS
