## Project Overview

**Aadrila AI** is a Next.js application that leverages artificial intelligence to provide intelligent solutions. This project demonstrates modern web development practices using the Next.js framework with TypeScript.

## Features

- Built with Next.js 14+ and React
- TypeScript for type safety
- Responsive design
- AI-powered functionality
- Optimized font loading with next/font

## Prerequisites

- Node.js 18.17 or later
- npm, yarn, pnpm, or bun package manager

## Setup Guide

### 1. Clone the Repository
```bash
git clone <repository-url>
cd aadrila-ai
```

### 2. Install Dependencies
```bash
npm install
# or
yarn install
# or
pnpm install
# or
bun install
```

### 3. Environment Configuration
Create a `.env.local` file in the root directory and add required environment variables:
```bash
# Add your configuration here
```

### 4. Run Development Server
```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure

```
aadrila-ai/
├── app/              # Next.js app directory
```

## Build for Production

```bash
npm run build
npm run dev
```

## Technologies Used

- **Framework:** Next.js 14+
- **Language:** TypeScript
- **Font:** Raleway (via next/font)
